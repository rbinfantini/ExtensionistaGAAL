TÍTULO DO PROJETO:

DESCRIÇÃO DO PROJETO:
Este projeto consiste em uma aplicação desenvolvida em Java (com interface gráfica utilizando a biblioteca Swing)
para demonstrar a aplicação prática de conceitos de Geometria Analítica e Álgebra Linear na área da Segurança da Informação. 

A ferramenta implementa a Cifra de Hill, utilizando operações com matrizes 2x2 (multiplicação de matrizes por vetores e
cálculo de matriz inversa em módulo 26) para encriptar e desencriptar blocos de texto.

O projeto foi desenvolvido como atividade extensionista para o curso de Ciência da Computação.

COMO EXECUTAR:
1. Compile todas as classes no BlueJ.
2. Clique com o botão direito na classe "AppCriptografia" e selecione "void main(String[] args)".
3. Insira a mensagem no campo de texto e clique em "Encriptar" ou "Desencriptar".
